Overview

This is a platform that aims to improve the hiring process make it more convenient for both the hiring managers and candidates.

Who This Platform Is For

This platform targets two types of users:

  • Hiring managers who are willing to hire software engineers. This platform enables a hiring manager to create assessments, invite candidates, and view the scores to compare performances in order to make hiring decisions.
  • Candidates invited to technical assessments by hiring managers.
